Hi there! Welcome to Kalail.


What is Kalail?

Kalail is my Django based experimental website. I say experimental but really I mean a testing ground for web-apps and the like. Every now and then I'm struck with an idea or a need. If I find that this tool doesn't exist, I set out to create it. Kalail is where I host these apps.


Can I use these apps?

Sure! Feel free to use any app you see on this site. Simply log in using Gmail, and as long as my server isn't overloaded, I'll continue to support any tools you use.


Who are you?

A software engineer and filmmaker. They go hand in hand. Really.
